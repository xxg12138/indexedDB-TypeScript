# indexedDB-TypeScript

# 这是一个基于 TypeScript 的 indexedDB ：

打包项目：

```
npx webpack
```

### 目录结构

```
├── dist                       // 项目打包目录
├── src                        // 源代码
│   ├── indexedDBTs            // indexedDB控制
│   │   ├── addIndexedDB       // 插入数据
│   │   ├── createIndexedDB    // 创建数据库
│   │   ├── index.ts           // 入口文件
│   └── index.html             // 主页
├── webpack.config             // webpack打包配置
└── package.json               // package.json
```
